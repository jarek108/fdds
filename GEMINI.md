# Gemini CLI Guidelines

## Experimentation
All temporary scripts, exploratory code, testing files, and scratchpads created by Gemini CLI MUST be placed inside the `playground/` directory.

This directory is strictly meant for prototyping and is ignored by git. Do not clutter the project root or source directories with temporary `temp_*.py` or `test_*.py` files unless they are intended to be permanent parts of the codebase.
